About EAuthz

EAuthz is a modern, secure authentication system built with Flask. It provides seamless login, registration, and session management to protect user data and enhance security.


Features include:

  • Password hashing with bcrypt
  • Persistent user database (SQLite)
  • Responsive, centered design for all pages
  • Smooth background gradients
